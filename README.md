# Amazon-Robot
Execução de cinco cenários automatizados de testes funcionais de interface do usuários com o Robot Framework.

#### [Vídeo com a execução dos cenários de teste](https://www.loom.com/share/caf4cf8bf6624030ba98ea4a079b255e)

### Para a execução do projeto em seu computador basta seguir os passos:
### Passo 1: Baixar o Python:
 [Link para download do Python](https://www.python.org/downloads/)

### Passo 2: Baixar o driver para seu navegador de preferência (Chrome ou Firefox), de acordo com a versão do mesmo
#### [Link para download do chromedriver](https://chromedriver.chromium.org/downloads)
#### [Link para download do geckodriver](https://github.com/mozilla/geckodriver/releases)
